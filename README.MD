# MOOC 自动互评（Tampermonkey 脚本）

一个用于中国大学 MOOC 作业互评页面的油猴脚本，支持自动打分、自动填写评语、白卷识别与自动流转下一份任务。

## 功能特性

- 自动识别可评分项并优先选择最高分
- 自动填写评语（从评语库中随机抽取）
- 白卷识别（检测内容/图片/附件），白卷时跳过并停止自动提交
- 可配置操作间隔（带随机浮动，降低机械操作特征）
- 支持自动提交与“继续评估下一份”自动跳转
- 必评任务完成后自动停止
- 悬浮控制面板（可拖拽、折叠、日志显示）
- 配置持久化（刷新页面后保留）

## 运行环境

- 浏览器：Chrome / Edge / Firefox（支持 Tampermonkey）
- 插件：Tampermonkey（油猴）
- 页面范围：`*://www.icourse163.org/learn/*`

## 安装方法

1. 安装 Tampermonkey 扩展。
2. 新建用户脚本。
3. 将 `mooc_auto_rate.user.js` 的内容完整粘贴进去并保存。
4. 打开中国大学 MOOC 课程的互评页面开始使用。

## 使用说明

1. 进入互评页面后，右侧会出现“互评助手”面板。
2. 点击“开始运行”后脚本开始循环执行：
   - 自动评分
   - 自动填评语
   - 满足条件时自动提交
   - 自动进入下一份（如存在按钮）
3. 点击“设置”可修改：
   - 随机评语库（每行一条）
   - 操作间隔（毫秒）
   - 是否自动提交
4. 点击“停止运行”可随时暂停。

## 配置项说明

- `随机评语库`：按行分割，脚本会随机选一条填入文本框
- `间隔(ms)`：基础延迟，实际执行会在约 `0.8x ~ 1.2x` 波动
- `自动提交`：开启后在检测完成且非白卷场景下自动点击提交

> 间隔会被限制在 `1 ~ 10000 ms` 的安全范围内。

## 行为说明（重要）

- 检测到疑似白卷时，脚本会提示并避免自动提交，需你手动确认。
- 若页面提示必评任务已完成，脚本会自动停止运行。
- 脚本依赖页面 DOM 结构；当 MOOC 页面改版后，可能需要更新选择器。

## 常见问题

### 1）面板没出现

- 确认 Tampermonkey 已启用该脚本
- 确认当前页面地址匹配 `icourse163.org/learn/`
- 尝试刷新页面

### 2）不自动提交

- 检查设置里是否开启“自动提交”
- 若存在白卷识别提示，属于保护行为，需要手动处理

### 3）评语不生效

- 请确认评语库非空，建议每行一条
- 页面文本框可能被课程页面逻辑重置，可适当增大间隔重试
